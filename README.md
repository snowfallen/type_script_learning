# Type script
### Install
```
npm i -g typescript
# or
yarn add global typescript
```
### Running 
```
tsc
```
### Comments for compiler
#### Disabling file verification
```
// @ts-nocheck
```
#### Enable file verification
```
// @ts-check
```
#### Ignore next line
```
// @ts-ignore
```
#### Waiting for an error on the next line
```
// @ts-expect-error
```
